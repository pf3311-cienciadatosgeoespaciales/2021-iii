Requisitos¶

Se requiere que los estudiantes cuenten con conocimientos sobre los siguientes temas:

  • Lenguaje de programación Python (o algún otro lenguaje de programación).

  • Interfaz de desarrollo Jupyter Notebook.

  • Lenguaje de marcado Markdown.

  • Sistema de control de versiones Git.

  • Plataforma de desarrollo colaborativo GitHub u otra similar (ej. GitLab, Bitbucket).

Además, es recomendable tener nociones de:

  • Distribución de Python Anaconda/Miniconda y gestor de paquetes Conda.

  • Ambientes de computación en la nube: Google Colab, Binder, Kaggle y otros.

  • Plataformas de alojamiento de sitios web: GitHub Pages, Netlify u otras.

  • Plataformas de alojamiento de aplicaciones web (data apps): Heroku, Streamlit Cloud u otras.

  • Estadística y análisis de datos.

  • Ciencia de datos.

  • Aprendizaje automatizado.
